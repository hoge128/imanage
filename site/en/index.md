## What it does

<ul class="points">
  <li><strong>Preserves creation time.</strong> File birth time (btime) is never modified.</li>
  <li><strong>Keeps JPG / RAW together.</strong> Matching RAW files and XMP sidecars are never left behind.</li>
  <li><strong>Reads XMP star ratings.</strong> From sidecar files or embedded XMP alike.</li>
  <li><strong>Safe deletion.</strong> Nothing is erased outright — it goes to the macOS Trash.</li>
  <li><strong>Japanese / English.</strong> The interface follows your system language.</li>
</ul>
