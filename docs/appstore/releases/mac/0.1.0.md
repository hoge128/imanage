# imanage 0.1.0 (Mac App Store)

初回リリース。

## Promotional Text

### 日本語

カメラで写真を撮影する人のための macOS アプリ。撮影日・機種・レンズ・評価などの EXIF をもとに、写真を思いどおりのフォルダ構成へ振り分けます。すべての処理は Mac の中だけで完結します。

### English

For photographers: sort photos into exactly the folders you want, straight from their EXIF — capture date, camera, lens, rating and more. Everything stays on your Mac.

## What's New

### 日本語

imanage の最初のリリースです。

・EXIF をもとに写真をフォルダへ振り分け
・振り分け後のフォルダ構成を実行前にツリーでプレビュー
・24 種類の階層フィールドを自由に並べ替え、プリセットとして保存
・⌘Z でいつでも取り消し
・ファイルの作成日時（btime）を保全
・RAW と XMP サイドカーをペアで移動

### English

The first release of imanage.

• Sorts photos into folders based on their EXIF
• Previews the resulting folder tree before anything moves
• 24 hierarchy fields you can reorder freely and save as presets
• Command-Z undoes any run
• File creation time (btime) is preserved
• RAW files and XMP sidecars move together with their JPG
