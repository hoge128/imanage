## できること

<ul class="points">
  <li><strong>撮影日時の保全。</strong>ファイルの作成日時（btime）を一切変更しません。</li>
  <li><strong>JPG / RAW をペアで扱う。</strong>同名の RAW と XMP サイドカーを取り残しません。</li>
  <li><strong>XMP の星評価に対応。</strong>サイドカー・埋め込みのどちらからも読み取ります。</li>
  <li><strong>安全な削除。</strong>完全削除はせず、必ず macOS のゴミ箱へ移動します。</li>
  <li><strong>日本語 / English。</strong>システムの言語に合わせて表示が切り替わります。</li>
</ul>

<div class="note">

Imanage はいかなる個人情報も収集・送信しません。写真の読み取りとフォルダの振り分けは、すべてお使いの Mac の中だけで完結します。ネットワーク通信も、解析ツールの組み込みもありません。
